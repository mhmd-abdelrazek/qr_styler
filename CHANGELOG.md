## 1.0.0

Initial stable release of QR Styler.

This version introduces a fully customizable QR code generator built with Flutter CustomPainter, allowing developers to create modern and branded QR codes with high flexibility.

Features included in this release:

- Customizable QR dot styling (color, size, and corner radius)
- Dot connection system (horizontal, vertical, none)
- Fully customizable finder eyes (outer, inner, pupil)
- Center logo support with safe zone protection
- Scalable QR rendering based on widget dimension
- High error correction support for reliable scanning
- Lightweight and performant implementation using CustomPainter
- Clean and simple API for easy integration in Flutter projects
